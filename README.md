# Days Counter

A web application built with Flask that provides various time calculation tools including date differences, countdown timers, age calculators, day of the week finders, and time elapsed calculators.

## Features

- **Date Difference Calculator**: Calculate the difference between two dates in days, weeks, months, and years.
- **Countdown Timer**: Set a target date and get a live countdown in days, hours, minutes, and seconds.
- **Age Calculator**: Calculate your exact age from a birth date in years, months, days, hours, minutes, and seconds.
- **Day of the Week Finder**: Find out what day of the week any date falls on.
- **Time Elapsed Calculator**: Calculate how much time has passed since a past date.
- **Responsive Design**: Works on desktop and mobile devices.
- **Dark Mode**: Toggle between light and dark themes.
- **AJAX Support**: Real-time updates without page refreshes.

## Technologies Used

- **Backend**: Flask (Python)
- **Frontend**: HTML5, CSS3, JavaScript
- **Icons**: Font Awesome
- **Date Handling**: Python's datetime and dateutil libraries

## Installation

1. Ensure you have Python 3.x installed on your system.
2. Install the required dependencies:
   ```
   pip install flask python-dateutil
   ```
3. Clone or download this repository.
4. Navigate to the project directory.

## Usage

1. Run the application:
   ```
   python app.py
   ```
2. Open your web browser and go to `http://127.0.0.1:5000/`
3. Select the desired calculation tool from the tabs.
4. Enter the required dates and click calculate.

## Project Structure

- `app.py`: Main Flask application with all routes and logic.
- `index.html`: Main HTML template with the user interface.
- `styles.css`: CSS styles for the application.
- `script.js`: JavaScript for interactivity and AJAX requests.
